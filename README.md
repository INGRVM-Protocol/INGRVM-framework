# INGRVM AI-Builder Framework
**The toolkit for building AI-First Ecosystems.**

The INGRVM Framework is a set of meta-tools and communication protocols designed for developers who build *with* and *for* AI. It contains the logic for cross-machine handoffs, automated journaling, and session recovery in an AI-driven development environment.

### Framework Components
1. **DAP (Distributed Authority Protocol):** A framework for multi-node Git management to prevent logic conflicts between machines.
2. **Mailroom:** A local P2P messaging service for machines to request file access or task handoffs.
3. **Session Protocol:** Automated checkpointing for long-running AI development sessions.
4. **The Journal:** A structured, AI-readable log of project intent and architectural decisions.

### Why INGRVM Framework?
If you use AI (like Gemini, Claude, or GPT) as your primary developer, you need a framework that manages the "Context" and "Intent" across multiple environments. INGRVM is that bridge.
